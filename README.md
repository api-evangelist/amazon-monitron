# Amazon Monitron (amazon-monitron)
Amazon Monitron is an end-to-end system that uses machine learning to detect abnormal behavior in industrial machinery, enabling you to implement predictive maintenance and reduce unplanned downtime. It includes sensors, a gateway, and the Monitron mobile app.
